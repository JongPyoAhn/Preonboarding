# Preonboarding
생각보다 갖다쓰는데 자신이 있던 나는 그 모든게 내 실력이 아닌것을 깨달아 버렸고 이번 프리온보딩을 통해 동시성의 근본을 공부하게 되었다.<br/>
학습자료를 업로드하면 안되기 때문에 공부하는 소스코드만 깃허브에 올리고 내용은 따로 노션에 정리!<br/>
